# BiliDow
Tools Downloder BiliBili Untuk Mendownload sebuah video dari aplikasi BiliBili.

# Command Termux
```python
pkg install git
pkg install python
git clone https://github.com/XnxcodeTeam/BiliDow
cd BiliDow
pip install requests
pip install pyfiglet
pip install rich
```
# Run Tools
```python
python bilibili.py
```
